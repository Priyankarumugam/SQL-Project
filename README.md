# 🛒 Amazon Sales Data Analysis (SQL Project)

## 📌 Overview

This project focuses on analyzing Amazon-like sales data using SQL. It includes database design, data insertion, and advanced queries to extract meaningful business insights.

---

## 🗂️ Database Structure

The project consists of 5 main tables:

* **Users** – Customer details
* **Orders** – Purchase transactions
* **Payments** – Payment information
* **Reviews** – Customer feedback
* **Returns** – Product return details

---

## ⚙️ Key Features

* Designed relational database with primary & foreign keys
* Performed **JOIN operations** across multiple tables
* Used **Aggregate Functions** (SUM, AVG, COUNT)
* Implemented **CTE & CASE statements**
* Created **Stored Procedure** for dynamic queries

---

## 📊 Key Insights

* Identified top customers based on total spending
* Analyzed product-wise performance and ratings
* Evaluated order trends and revenue
* Tracked return behavior and customer feedback

---

## 🛠️ Tools Used

* SQL Server
* T-SQL

---

## 🚀 Conclusion

This project demonstrates strong SQL skills including data modeling, querying, and business analysis, making it suitable for Data Analyst roles.
